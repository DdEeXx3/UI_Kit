For creating this table we used classes:

  • "text-center" - text inside every cell is centered,
  • "table-layout-80-fixed-center" - table is 80% width, centered in div with fixed text inside cells,
  • "table-medium-radius" - it has medium (1em) radius of borders,
  • "table-border-1-solid" - it has 1px width solid border,
  • "table-small-padding" - there is small padding inside table (15px for caption, 10px for header and 5px for cell),
  • "table-color-default" - colors of table elements have default color saturation.
  • "medium" class was additionally added to section element to have its text bigger.
