Forms

Class nameDescription
input-$size-$widthSets size of form input:

$size - value describing size of an input.
This parameter gets 3 values:
-small (with 9px - 11px font size)
- medium (with 11px - 16px font size)
-big (with 16px - 24px font size)

$width - percentage value of input width in reference to div, in which this input has been created.
This parameter gets values from 1 to 100 or default
input-$radiusValueSets radius level of a button:

$radiusValue - value describing radius level of a button.
This parameter gets 5 values:
- no-radius (with 0 border radius)
- small-radius (with 0.5em border radius)
- medium-radius (with 1em border radius)
- big-radius (with 2em border radius)
- oval (with 50% border radius)
input-$style-$colorSets style and default color of form input:

$style - number of style you want to choose.
This parameter gets 2 values:
-style1 (with border-bottom only)
-style2 (with border all around the input)

$color - default color style of your input. This parameter gets 6 values:
- primary (first main page color, should be light)
- secondary (second main page color, should be dark)
- teritary (third main page color, it may be any color)
- success (color responsible for successes and any positive actions on your site)
- error (color responsible for errors and any negative actions on your site)
- warning (color resposible for displaying any warning on your site)
input-$element-$colorSets any input element's color:

$element - any input's element you want to have color changed.
This parameter gets 3 values:
-font (for changing base color of a font)
-placeholder (for changing base color of a placeholder)
-background (for changing base color of a background)

-$color - a color on which you want to change your element.
This parameter gets 6 values of $color variable (as in example above)
